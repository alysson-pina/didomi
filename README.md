# didomi-frontend-challenge

## Features

* React 16
* Webpack 4
* Babel 7
* Hot Module Replacement

## DIY Add-Ons

* [ESLint]
* [CSS Modules]
* [SVG Icons]
* [Fonts Support]
* [Images Support]


## Installation

* `git clone git@github.com:alysson-pina/didomi.git`
* cd didomi
* npm install
* npm start
* visit `http://localhost:8080/`
